# PJAMASC Website

## Installation Instructions
1. Install Apache 2.
	1b. Set the document root to `<project_root>/public`.
2. Install PHP version 7.2 or higher.
3. Install MySQL.
	3b. Run `pjamasc.sql`.
4. Install [Composer](https://getcomposer.org/download/).
5. Run `composer install` in the project root.
